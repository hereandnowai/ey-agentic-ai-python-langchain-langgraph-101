## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/hereandnowai/ey-agentic-ai-python-langchain-langgraph-101.git
cd ey-agentic-ai-python-langchain-langgraph-101

# Create a virtual environment
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate

# Install dependencies (added during the sessions)
pip install -r requirements.txt
```

Copy `.env.example` to `.env` and add your API keys before running any LLM examples.
